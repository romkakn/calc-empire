# Creatinine Clearance (Cockcroft-Gault) Research

## Formula
**Cockcroft-Gault Equation:**
- **Male:** CrCl = ((140 - Age) × Weight_kg) / (72 × Serum_Creatinine_mg_dL)
- **Female:** CrCl = [((140 - Age) × Weight_kg) / (72 × Serum_Creatinine_mg_dL)] × 0.85

**Units:**
- Age: years
- Weight: kg
- Serum Creatinine: mg/dL (Note: If in µmol/L, divide by 88.4)
- Result: mL/min

**Weight Considerations (Advanced):**
- Use **Actual Body Weight (ABW)** if ABW < Ideal Body Weight (IBW).
- Use **Ideal Body Weight (IBW)** if ABW > IBW but < 1.3 × IBW.
- Use **Adjusted Body Weight (AjBW)** if ABW > 1.3 × IBW (obese).
- *IBW (Male):* 50kg + 2.3kg for each inch over 5ft.
- *IBW (Female):* 45.5kg + 2.3kg for each inch over 5ft.
- *AjBW:* IBW + 0.4 × (ABW - IBW).
